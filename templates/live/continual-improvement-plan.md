# Purpose

Plan ongoing service improvements based on evidence, priorities and capacity.

# When to use this

Use throughout Live to manage improvements beyond routine support.

# Inputs

- Service dashboard
- User feedback
- Service reviews
- Backlog and roadmap

# Template

Add improvement opportunity, evidence, priority, owner, expected outcome, effort, status and review date.

# Quality checklist

- Improvements are evidence based.
- Priorities are transparent.
- Outcomes are measurable where possible.
