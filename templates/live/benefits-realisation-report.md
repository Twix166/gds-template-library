# Purpose

Track realised benefits against expected outcomes, measures and evidence.

# When to use this

Use after launch and at agreed benefit review points.

# Inputs

- Benefits register
- Service dashboard
- Business case
- Operational and financial data

# Template

Add expected benefit, measure, baseline, target, current result, evidence, owner and next action.

# Quality checklist

- Benefits are measured against baselines where available.
- Variance is explained.
- Actions are linked to improvement planning.
