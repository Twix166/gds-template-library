# Purpose

Summarise Discovery findings, evidence, recommendations and proposed next steps.

# When to use this

Use at the end of Discovery or for major review points.

# Inputs

- Research findings
- Service data
- Technical and operational findings
- Options assessment

Copy this structure for an end-of-Discovery report or playback document.
