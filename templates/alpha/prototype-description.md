# Purpose

Describe what a prototype is intended to test and how it supports learning.

# When to use this

Use for each significant prototype or prototype iteration in Alpha.

# Inputs

- Hypotheses
- User needs
- Journey maps
- Research plan

# Template

Add prototype scope, assumptions, user journeys, content, technical approach, test plan and learning goals.

# Quality checklist

- The prototype has clear research questions.
- It is only as detailed as needed to learn.
- Known limitations are documented.
